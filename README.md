# gobo.cf
URL Shortener For Scratch Projects

| URL           | Function           |
| ------------- | ------------------ |
| /u/?to=       | To User Page       |
| /s/?to=       | To Studio Page     |
| /p/?to=       | To Project Page    |
| /p/e/?to=     | To Project Editor  |
| /d/?to=       | To Topic           |
